# Destination Engine
It is a complete OpenGL 4.2+ engine designed for developing small games and interactive rendering scenes, featuring real-time rendering and physics simulation.

## Features
- Blinn-Phong lighting model with support for directional, point, and spot lights
- Deferred renderer with light volume optimization
- Off-screen scene rendering through user-defined camera render targets
- Materials system supporting albedo, normal and specular mapping
- HDR rendering, gamma correction and fog
- Resource system for loading 3D models, textures, and material creation
- Component-based game object architecture
- Integrated physics system featuring colliders (box, capsule), rigid bodies, and ray casting
